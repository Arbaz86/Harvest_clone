Harvest Website Clone
=====================

GetHarvest is a powerful web application designed to streamline time tracking and project management. As part of a talented team of 4, I was responsible for creating the front-end and back-end of the Social Sign-in/Sign-up page, contributing to the website design, and developing a landing page. Our Tech stack included **_HTML, CSS, JavaScript, React, Redux, Chakra UI, Node.js, Express.js, MongoDB, and Render_** . Within a short time frame of 6 days, we successfully developed and delivered a functional and efficient project management app with a sleek design and user-friendly interface

Features
--------

*   Fully responsive design that looks great on all devices.
*   User account authentication system with login and logout functionality.
*   Ability to add and remove social media channels for each project.
*   Project management system that allows users to create, edit, and view projects by date.
*   Comprehensive analytics reporting that provides insights on project performance.
*   Invoice management system that lets users view and manage project invoices.
*   Team management functionality that allows users to add and remove team members.
*   Welcome email sent to users when they sign in to the website.
*   Password reset feature that allows users to reset their password if they forget it.

Getting Started
---------------

This web application for online social media marketing management was built using React, Redux, Node.js, Express, MongoDB, Chakra UI, CSS, JavaScript, and HTML. To run the application on your local environment, please follow these guideline

### Prerequisites

*   React
*   Redux
*   Node.js
*   MongoDB
*   Express.js
*   HTML
*   CSS

### Setup

```
git clone https://github.com/Arbaz86/Harvest_clone.git
```

### Install

To install all the dependences of the project, run the following command:

```
npm install --force
```

To run the application got to the client folder and run the following command:

```
npm run dev
```
